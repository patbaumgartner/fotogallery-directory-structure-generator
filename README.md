# fotogallery-directory-structure-generator

A Spring Boot command-line tool that reads a gallery CSV file (generated by [foto-gallery-qrcode-generator](https://github.com/patbaumgartner/foto-gallery-qrcode-generator)) and creates a ready-to-use portrait directory structure on disk.

## Directory Structure

For each gallery entry in the CSV the tool creates:

```
{Event Name}/
└── portraits/
    ├── {gallery-code-1}/
    ├── {gallery-code-2}/
    └── ...
```

For example, given an event named `SchoolEvent2024` with two gallery codes:

```
SchoolEvent2024/
└── portraits/
    ├── ABCD-EFGH-IJKL/
    └── MNOP-QRST-UVWX/
```

## Prerequisites

| Requirement | Version |
|-------------|---------|
| Java (JDK)  | 17+     |
| Maven       | 3.9+    |

## Getting Started

### Build from source

```bash
# Clone the repository
git clone https://github.com/patbaumgartner/fotogallery-directory-structure-generator.git
cd fotogallery-directory-structure-generator

# Build (skipping tests for speed)
mvn clean package -DskipTests

# Build including tests
mvn clean verify
```

The fat JAR is produced at `target/fotogallery-directory-structure-generator-*.jar`.

## Usage

```bash
java -jar target/fotogallery-directory-structure-generator-*.jar \
  --app.csv-input-path=codes.csv \
  --app.output-path=./output
```

### Configuration

| Property              | Default      | Description                                                              |
|-----------------------|--------------|--------------------------------------------------------------------------|
| `app.csv-input-path`  | `codes.csv`  | Path to the CSV file generated by foto-gallery-qrcode-generator          |
| `app.output-path`     | `.`          | Base directory where the event directory structure will be created        |

All properties can also be set via environment variables or an external `application.properties` file.

### CSV Format

The tool reads the CSV format produced by [foto-gallery-qrcode-generator](https://github.com/patbaumgartner/foto-gallery-qrcode-generator):

```
Number,Code,Password,Event Name,URL
1,ABCD-EFGH-IJKL,password123,SchoolEvent2024,https://example.com/gallery?code=ABCD-EFGH-IJKL
2,MNOP-QRST-UVWX,password456,SchoolEvent2024,https://example.com/gallery?code=MNOP-QRST-UVWX
```
